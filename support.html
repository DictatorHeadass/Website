Support - Still Waters SMP

Welcome to the Still Waters SMP support page! If you’re experiencing any issues or have questions, you’ve come to the right place.

Common Issues

  • Server Connection Problems: Make sure you are using the correct IP and port. Ensure your Minecraft client version matches the server Forge version 1.20.4.
  • Mod Compatibility: Our modpack is carefully curated, but sometimes mods conflict. Try launching with a minimal mod list if you experience crashes.
  • Performance Troubleshooting: Check your allocated RAM and try optimizing your client or server configs for better performance.

How to Get Help

If your problem isn’t covered here, please:

  • Join our Discord server https://discord.gg/9Fk77mjnzt
  • Check the FAQ on the About page
  • Send us an email at wanderworldsmc@gmail.com
